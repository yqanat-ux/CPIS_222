# CPIS-222: CPU Scheduling Algorithms

This repository contains implementations of fundamental CPU process scheduling algorithms. The project focuses on visualizing how operating systems manage process execution and handle CPU allocation to maximize efficiency.

**Author:** Yasser Qanat

## 🚀 Overview
The core objective of this project is to simulate how different scheduling algorithms affect process wait times, turnaround times, and overall CPU throughput. It provides a clear implementation of how these algorithms handle process queues and context switching.

## 🛠️ Algorithms Implemented

*   **First-Come, First-Served (FCFS):** The simplest scheduling algorithm where processes are executed in the exact order they arrive in the ready queue.
*   **Shortest Job First (SJF):** A non-preemptive algorithm that selects the process with the smallest execution time to minimize the average waiting time.
